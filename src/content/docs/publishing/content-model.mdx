Blog posts require `title`, `description`, and `publishDate`. Docs require `title` and `description`, and can use `order` to control docs listing order.

Shared fields include:

- `updatedDate`
- `tags`
- `draft`
- `comment`
- `heroImage`
- `accentColor`

Production builds exclude drafts. Development builds show drafts so writing can happen in place.
